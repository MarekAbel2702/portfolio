Privacy Policy – Last Round

Last updated: December 18, 2025

Your privacy is important to us. This Privacy Policy explains how the Last Round application handles data and protects user privacy.

1. Data collection

The Last Round application does not collect, store, or transmit any personal data. The application does not require user accounts and does not request any information that could identify the user.

2. App content

Last Round is an application that provides a locally available list of drink recipes and related information. The drink database is stored within the application and is used only to display content to the user.

The application does not collect information about:

  • which drinks the user views
  • which recipes the user searches for
  • how the user interacts with the application
  • any preferences or personal choices

3. Local operation

Last Round works locally on the user’s device. The application does not require an internet connection to access its built-in drink database.

All application content is:

  • stored locally within the application
  • used only to display drink recipes and information
  • not connected to any user profile
  • not transmitted to any external server

4. Location data

The application does not access GPS or location services. Last Round does not determine, collect, store, or share the user’s physical location.

5. Ads and analytics

The application:

  • does not display advertisements
  • does not use analytics tools
  • does not integrate third-party tracking SDKs
  • does not track users

6. App permissions

Last Round does not request any dangerous or sensitive permissions. The application does not access:

  • location
  • camera
  • microphone
  • contacts
  • storage or files
  • network data for tracking purposes

7. Data sharing

Since no personal data is collected, Last Round does not share any user data with third parties.

8. Data security

Last Round processes its content locally on the user’s device. No personal data is transmitted over the internet, minimizing any risk of data exposure.

9. Age restrictions and children’s privacy

Last Round contains content related to alcoholic drink recipes. The application is not intended for children and should only be used by users who are legally allowed to access alcohol-related content in their country or region.

Last Round does not knowingly collect any data from children or minors.

10. Responsible use

The content in Last Round is provided for informational purposes only. Users are responsible for complying with local laws and regulations related to alcohol consumption. The application does not encourage irresponsible or underage drinking.

11. Changes to the Privacy Policy

This Privacy Policy may be updated from time to time. Any changes will be published with an updated revision date.
