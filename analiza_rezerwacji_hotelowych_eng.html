Hotel Reservation Analysis

Data analysis and visualization in Power BI

Project Overview

This project aimed to analyze hotel reservation data to understand trends in room occupancy, revenue, and customer reviews. Key questions addressed in the analysis:

  • 📌 When does the hotel experience the highest and lowest occupancy?
  • 📌 Which room types are most frequently booked and rated highest by customers?
  • 📌 How do room prices vary across different months?
  • 📌 What are the most common payment methods used by guests?
  • 📌 What actions can be taken to increase revenue and improve services?

Technologies Used

  • SQL
  • T-SQL
  • Power BI

Scope of Analysis

The project involved creating and analyzing a relational database, followed by data visualization in Power BI. The following key components were utilized:

  • ✅ SQL / T-SQL: Database structure design, data insertion scripts, views, stored procedures, and query optimization.
  • ✅ Power BI: Development of dashboards to analyze hotel occupancy, finances, and customer reviews.

Key Findings & Conclusions

  • ✅ Peak occupancy: June, July, December, and April.
  • ✅ Presidential Suite received the highest rating (5.0), while Single Rooms had the lowest rating (3.86).
  • ✅ Room prices fluctuate seasonally – dynamic pricing strategies should be implemented.
  • ✅ Most common payment methods: bank transfers and credit/debit cards.
