System Analizy Sprzedaży

Analiza sprzedaży w Python

Opis Projektu

Stworzyłem kompletny system do analizy danych sprzedażowych, łącząc backend SQL Server z analizą w Pythonie. Projekt zawiera strukturę relacyjnej bazy danych, zabezpieczenia, procedury składowane oraz raporty i wykresy przygotowane z użyciem bibliotek pandas, matplotlib i seaborn.

Użyte Technologie

  • SQL
  • T-SQL
  • Python
  • Matplotlib
  • Numpy
  • Pandas
  • Seaborn
  • Git

📝 Zakres prac:

  • 🗄️ Baza danych SQL Server – tabele orders, products, customers, stores z relacjami i danymi testowymi
  • 🔐 Bezpieczeństwo danych – Dynamic Data Masking (DDM), role użytkowników, nadawanie uprawnień (UNMASK)
  • ⚙️ Logika biznesowa – procedury składowane (GetTopProductsByRevenue, InsertNewCustomer, GetMonthlySales), triggery (PreventHighDiscount, LogOrdersInsert)
  • 📊 Analiza w Pythonie – połączenie z bazą, analiza danych z użyciem pandas i numpy, przekształcanie dat, agregacje, grupowanie
  • 📈 Wizualizacje – wykresy słupkowe, kołowe, heatmapy korelacji (matplotlib, seaborn)
  • 🧠 Zastosowanie biznesowe – analiza przychodów, efektywność rabatów, trendy miesięczne, udział kategorii i płci
