Sales Analytics System

Sales analytics with Python

Project Description

I created a complete system for analyzing sales data by combining a SQL Server backend with data analysis in Python. The project includes a relational database structure, data security features, stored procedures, as well as reports and visualizations prepared using pandas, matplotlib, and seaborn libraries.

Technologies Used

  • SQL
  • T-SQL
  • Python
  • Matplotlib
  • Numpy
  • Pandas
  • Seaborn
  • Git

📝 Project Scope:

  • 🗄️ SQL Server database – tables: orders, products, customers, stores with relations and test data
  • 🔐 Data security – Dynamic Data Masking (DDM), user roles, permission management (UNMASK)
  • ⚙️ Business logic – stored procedures (GetTopProductsByRevenue, InsertNewCustomer, GetMonthlySales), triggers (PreventHighDiscount, LogOrdersInsert)
  • 📊 Python analysis – database connection, data analysis with pandas and numpy, date transformation, aggregation, grouping
  • 📈 Visualizations – bar charts, pie charts, correlation heatmaps (matplotlib, seaborn)
  • 🧠 Business applications – revenue analysis, discount efficiency, monthly trends, category and gender distribution
