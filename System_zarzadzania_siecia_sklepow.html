Baza danych MS SQL

System zarządzania siecią sklepów

Opis Projektu

Zbudowałem kompletną bazę danych w T-SQL odwzorowującą rzeczywisty system sprzedażowo-magazynowy dla sieci sklepów detalicznych.

Projekt obejmuje:

  • ✅Model relacyjny (sklepy, pracownicy, produkty, magazyn, zamówienia, zwroty)
  • ✅Widoki, funkcje, procedury składowane i triggery biznesowe
  • ✅Dane testowe – sprzedaż, dostawy, zatrudnienia
  • ✅System ról i uprawnień z kontrolą dostępu (RBAC)
  • ✅Modułowe skrypty .sql w repozytorium GitHub
  • ✅Całość wykonana wyłącznie w T-SQL (bez aplikacji frontend/backend)
  • ✅System zabezpieczony – 3 role użytkowników z ograniczonymi dostępami
