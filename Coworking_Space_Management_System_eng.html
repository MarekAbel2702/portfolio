PostgreSQL Database

Coworking Space Management System

Project Description

I developed a comprehensive PostgreSQL database project for managing a coworking space.

The system allows for:

  • ✅User registration
  • ✅Desk and conference room reservations
  • ✅Payment processing and automatic invoice generation
  • ✅Analytics and reports (revenue, resource utilization)
  • ✅Roles and security: separate access for client and administrator
  • ✅Views, PL/pgSQL functions, triggers

Technologies Used

  • PostgreSQL
  • SQL
  • T-SQL
  • Git
