Log Monitoring System

Application log monitoring and analysis system

Project Description

This project presents a complete system for analyzing data from log files. The solution enables automatic processing of application logs saved in a text file, structured storage in a relational database, and exporting and presenting the data in various formats and visual forms. The system consists of several independent modules responsible for individual stages: data parsing, database communication, export, and visualization. The project was designed with a focus on clarity, scalability, and practical application.
