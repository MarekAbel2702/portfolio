REST API in ASP.NET Core with Onion Architecture

OnionProductAPI

Project Description

A backend project based on the Onion architecture, built with ASP.NET Core 8 and C#. The application provides a REST API for managing products, with full CRUD support and integration with a Microsoft SQL Server database using Entity Framework Core.
