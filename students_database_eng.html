Students Database

Application for managing student data

Project Description

Students Database is a web application created with ASP.NET MVC technology, allowing management of student data. Users can add, edit, and delete students, as well as view the list of students stored in an SQL database.

Technologies Used

  • C#
  • .NET
  • SQL
  • Entity Framework
  • ASP.NET MVC
