Privacy Policy – CompassBuddy

Last updated: December 18, 2025

Your privacy is important to us. This Privacy Policy explains how the CompassBuddy application handles data and ensures user privacy.

1. Data collection

The CompassBuddy application does not collect, store, or transmit any personal data. The application does not require user accounts and does not request any information that could identify the user.

2. Sensor usage

CompassBuddy uses device sensors (such as the magnetometer and accelerometer) solely to determine direction and display compass information.

All sensor data is:

  • processed locally on the user’s device
  • used only in real time
  • not stored
  • not transmitted to any server

3. Location data

The application does not access GPS or location services. CompassBuddy determines direction using internal device sensors only and does not determine the user’s physical location.

4. Ads and analytics

The application:

  • does not display advertisements
  • does not use analytics tools
  • does not integrate third-party SDKs
  • does not track users

5. App permissions

CompassBuddy does not request any dangerous or sensitive permissions. The application does not access:

  • location
  • camera
  • microphone
  • contacts
  • storage or files
  • network data

6. Data sharing

Since no personal data is collected, CompassBuddy does not share any data with third parties.

7. Data security

All data processing occurs locally on the user’s device. No data is transmitted over the internet, minimizing any risk of data exposure.

8. Children’s privacy

CompassBuddy is not specifically intended for children and does not knowingly collect any data from minors.

9. Changes to the Privacy Policy

This Privacy Policy may be updated from time to time. Any changes will be published with an updated revision date.
