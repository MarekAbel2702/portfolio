MS SQL Database

Retail Chain Management System

Project Description

I built a complete T-SQL database reflecting a real sales and warehouse system for a retail store chain.

The project includes:

  • ✅Relational model (stores, employees, products, warehouse, orders, returns)
  • ✅Views, functions, stored procedures, and business triggers
  • ✅Test data – sales, deliveries, employment
  • ✅Role-based access control system (RBAC)
  • ✅Modular .sql scripts in GitHub repository
  • ✅Entirely implemented in T-SQL (no frontend/backend application)
  • ✅Secured system – 3 user roles with restricted access

Technologies Used
