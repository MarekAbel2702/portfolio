Privacy Policy – SunTimes

Last updated: January 2026

Your privacy is important to us. This Privacy Policy explains how the SunTimes application handles data and protects user privacy.

1. Data collection

SunTimes does not collect, store, or transmit any personal data. The application does not require user accounts and does not collect information that could identify users.

2. Location data (GPS)

SunTimes uses the device’s location (GPS) only to determine the user’s current location in order to calculate accurate sunrise and sunset times.

  • location data is processed locally on the device
  • location data is used only while the app is running
  • location data is not stored
  • location data is not transmitted to any server

3. Offline functionality

SunTimes works fully offline. All calculations for sunrise, sunset, and Moon phases are performed locally on the device.

4. Moon phases and astronomical data

The application provides information about Moon phases and solar events. This data is calculated locally and does not require internet access.

5. Ads and analytics

The application:

  • does not display advertisements
  • does not use analytics tools
  • does not track users
  • does not use third-party tracking SDKs

6. App permissions

SunTimes requests only the permissions necessary for its functionality.

  • Location – to calculate sunrise and sunset times

The application does not access:

  • camera
  • microphone
  • contacts
  • files or storage
  • network data

7. Data sharing

SunTimes does not share any data with third parties. No personal or location data is collected or transmitted.

8. Data security

All processing occurs locally on the user’s device. Since no data is transmitted or stored, the risk of data exposure is minimized.

9. Children’s privacy

SunTimes does not knowingly collect any data from children. The application is safe to use for all age groups.

10. Changes to this Privacy Policy

This Privacy Policy may be updated in the future. Any changes will be reflected by an updated revision date.
